# Proyecto: Notifications Page

## Descripción del Proyecto

Este proyecto es una página de notificaciones interactiva que muestra un contador de notificaciones no leídas y permite marcar todas las notificaciones como leídas con un solo clic. Cada notificación tiene información como el nombre del usuario, el tipo de interacción (comentario, mensaje privado, etc.), y la fecha en que ocurrió. Además, las notificaciones tienen un diseño visual atractivo, donde se marca con un fondo destacado aquellas que están activas (no leídas).

## Características

- **Contador de notificaciones**: Muestra cuántas notificaciones no leídas hay en tiempo real.
- **Marcar todas como leídas**: Permite al usuario marcar todas las notificaciones como leídas con un solo clic.
- **Interacción con notificaciones**: El usuario puede interactuar con las notificaciones para cambiarlas de estado (de leída a no leída).
- **Diseño atractivo**: Utiliza un diseño limpio y moderno con colores suaves y agradables, y con interacciones visuales claras.
- **Responsive**: La página se adapta bien a diferentes tamaños de pantalla para mejorar la experiencia de usuario en dispositivos móviles y de escritorio.

## Desarrollo del Challenge

Este desafío fue desarrollado utilizando HTML, CSS y JavaScript. A continuación se detallan las tecnologías y bibliotecas utilizadas:

- **HTML**: Estructura básica de la página de notificaciones.
- **CSS**: Diseño y estilos visuales de la página, incluyendo el layout y los efectos de las notificaciones.
- **JavaScript**: Funcionalidad para manejar la interacción con las notificaciones y marcar todas las notificaciones como leídas.

### Funcionalidades implementadas:

- **Contador de notificaciones**: Muestra el número de notificaciones no leídas en la parte superior de la página.
- **Marcar todas como leídas**: Al hacer clic en el botón "Mark all as read", todas las notificaciones se marcan como leídas y se actualiza el contador.
- **Interacción individual con notificaciones**: Cada notificación puede ser marcada como leída al hacer clic sobre ella.
- **Notificaciones activas**: Las notificaciones no leídas se resaltan visualmente para indicar su estado.
